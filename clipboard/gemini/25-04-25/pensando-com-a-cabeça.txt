É importante que você tente entender cada ponto abaixo como uma etapa do meu pensamento. Vale ressaltar que o sistema deve se igualar ao meu fluxo de pensamento, e não o contrário. 


1. Recebe o conteúdo extraído.
2. Percorre o conteúdo recebido procurando uma string correspondente a {{TOPICO}}.
3. Continua percorrendo o conteúdo em busca de uma string correspondente a {{/TOPICO}}.
4. Pronto! O primeiro tópico foi encontrado.
5. Armazenar em um array ou objeto o conteúdo do primeiro tópico.
6. Continuar varrendo o conteúdo extraído buscando pelos itens 2 e 3 acima. Isso fará com que o sistema busque por cada um dos tópicos. E uma vez que encontrar, os armazenará em um array/objeto. 
Se, após encontrar uma string correspondente a {{/TOPICO}} o sistema não encontrar uma próxima string correspondente a {{TOPICO}}, o sistema considerará como lixo o que encontrar depois da última string {{/TOPICO}}. Assim garantimos que todo o conteúdo da aula estará dentro do par de tags {{TOPICO}}/{{/TOPICO}}.
7. Pega o primeiro item do array/objeto (topico).
8. Procura uma string correspondente a {{SECAO}}.
9. Continua percorrendo o conteúdo em busca de uma string correspondente a {{/SECAO}}.
10. Pronto! A primeira seção do primeiro tópico foi encontrada.
11. Armazena o conteúdo dela em um array/objeto Seções.
12. Faz isso para cada uma das seções(repetição dos itens 8 e 9).
13. Tudo que estiver fora do par {{SECAO}}/{{/SECAO}} mas mesmo assim, encapsulado no par maior {{TOPICO}}/{{/TOPICO}} é imediatamente descartado pois é lixo.
Dessa forma, conseguimos 'cercar' todo o conteúdo que importa e descartar o que não nos serve.
